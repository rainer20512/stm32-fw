Steps to make LVGL run                                           RHB / 11.06.25
-------------------------------------------------------------------------------

000 As third party tools rely on that, the name of the subdir, lvgl resides in, 
    has to be 'lvgl'!
     
001 As describe in setup guide, copy lv_conf-Template.h to lv_conf.h and chng
    #if 0 to #if 1

002 in lv_conf_internal.h in the scr subdir, remove one ../ from the line
           #include "../../lv_conf.h"                /* Else assume lv_conf.h is next to the lvgl folder. */
    so it reads
           #include "../lv_conf.h"                /* Else assume lv_conf.h is next to the lvgl folder. */

003 in lv_font_montserrat_14_aligned.c : change
      #include "lvgl/lvgl.h"
    to read
      #include "../../lvgl/lvgl.h"

004 in lv_conf.h: set LV_USE_LOG to 1

005 The whole lvgl-tree should compile now w/o any errors or warnings

006 connect tick interface: 
    a) in stm32xxx_it.c in SysTick_Handler: 
       insert these three lines:
         #include "../../lvgl/lv_conf.h"
         lv_tick_inc(1);
         TaskNotify(TASK_LVGL);
    b) in main.c/CheckForSleep:
       comment out the code lines manipultating SysTick
          // HAL_SuspendTick();
       and below
          // HAL_ResumeTick();
    c) set CPU frq to a sufficient value, ie >= 24MHz to make (re)draw efficient 
007 Enable GC9A01 Driver:
    a) Find this line in lv_conf: 
        #define LV_USE_ST7735        0
    b) after that define block add
        #define LV_USE_GC9A01        1 
    c) and also expand the or-clause at the following #if by
         ... | LV_USE_GC9A01 ...
    d) after the #if block add the following:
        #if LV_USE_GC9A01
            /* 
             * GC9A01 driver needs this set to 1
             */
             #define LV_COLOR_16_SWAP  1 
        #endif
    e) Find this line in src/lv_conf_internal.h
      #if (LV_USE_ST7735 | LV_USE_ST7789 | LV_USE_ST7796 | LV_USE_ILI9341 .....
    f) add  ... | LV_USE_GC9A01 ... to the or'ed expression
    g) unter src/drivers/display den komletten ordner gc9a01 ( 6 files ) ablegen

008 Enable XML runtime Support
    a) in lv_conf.h:
        set LV_USE_OBJ_NAME to 1
        set LV_USE_XML to 1
    b) in xmlparse.c in getDebugLevel ( ganz unten ):
